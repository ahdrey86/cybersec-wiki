# Кабели и физический уровень

Среды передачи данных и их характеристики.

## Витая пара (Twisted Pair)

Наиболее распространённый тип кабеля для LAN.

### Категории

| Категория | Частота | Скорость | Применение |
|-----------|---------|----------|------------|
| Cat 5 | 100 МГц | 100 Мбит/с | Устаревший |
| Cat 5e | 100 МГц | 1 Гбит/с | Домашние сети |
| Cat 6 | 250 МГц | 1 Гбит/с (10G до 55м) | Офисы |
| Cat 6a | 500 МГц | 10 Гбит/с | Дата-центры |
| Cat 7 | 600 МГц | 10 Гбит/с | Экранированные |
| Cat 8 | 2000 МГц | 25-40 Гбит/с | Серверные |

### Типы экранирования

```
UTP  — Unshielded Twisted Pair (неэкранированная)
FTP  — Foiled Twisted Pair (общий экран из фольги)
STP  — Shielded Twisted Pair (экран каждой пары)
S/FTP — Общий экран + экран каждой пары
```

### Разъём RJ-45

**Стандарты обжима:**

| T568A | T568B |
|-------|-------|
| 1. Бело-зелёный | 1. Бело-оранжевый |
| 2. Зелёный | 2. Оранжевый |
| 3. Бело-оранжевый | 3. Бело-зелёный |
| 4. Синий | 4. Синий |
| 5. Бело-синий | 5. Бело-синий |
| 6. Оранжевый | 6. Зелёный |
| 7. Бело-коричневый | 7. Бело-коричневый |
| 8. Коричневый | 8. Коричневый |

:::info Прямой vs Кроссовер
- **Прямой** (T568B—T568B): компьютер ↔ коммутатор
- **Кроссовер** (T568A—T568B): компьютер ↔ компьютер

Современное оборудование с Auto-MDI/X определяет тип автоматически.
:::

## Оптоволокно (Fiber Optic)

Передача данных световыми импульсами.

### Типы волокна

| Тип | Диаметр сердцевины | Дальность | Применение |
|-----|-------------------|-----------|------------|
| **Многомодовое (MMF)** | 50/62.5 мкм | до 2 км | Внутри здания |
| **Одномодовое (SMF)** | 9 мкм | до 100+ км | Магистрали |

### Разъёмы

| Разъём | Описание |
|--------|----------|
| **SC** | Square Connector, push-pull |
| **LC** | Lucent Connector, компактный |
| **ST** | Straight Tip, байонетный |
| **FC** | Ferrule Connector, резьбовой |
| **MPO/MTP** | Multi-fiber, 12-24 волокна |

### Стандарты оптических трансиверов

| Стандарт | Скорость | Интерфейс |
|----------|----------|-----------|
| SFP | 1 Гбит/с | LC |
| SFP+ | 10 Гбит/с | LC |
| QSFP+ | 40 Гбит/с | MPO |
| QSFP28 | 100 Гбит/с | MPO |

## Коаксиальный кабель

Устаревший для LAN, используется в кабельном ТВ и некоторых WAN.

| Тип | Импеданс | Применение |
|-----|----------|------------|
| RG-6 | 75 Ом | Кабельное ТВ, интернет |
| RG-58 | 50 Ом | Старые сети 10Base2 |
| RG-59 | 75 Ом | Видеонаблюдение |
