# Асимметричное шифрование

Асимметричные алгоритмы используют **пару ключей**: открытый (публичный) и закрытый (приватный).

## Основные алгоритмы

### RSA

Основан на сложности факторизации больших чисел.

| Параметр | Рекомендации |
|----------|--------------|
| Размер ключа | ≥ 2048 бит (рекомендуется 4096) |
| Применение | Шифрование, ЭЦП, обмен ключами |

```python
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# Генерация ключей
key = RSA.generate(4096)
public_key = key.publickey()

# Шифрование
cipher = PKCS1_OAEP.new(public_key)
ciphertext = cipher.encrypt(b"secret")
```

### Эллиптические кривые (ECC)

Основаны на задаче дискретного логарифма в группе точек эллиптической кривой.

| Кривая | Размер | Эквивалент RSA |
|--------|--------|----------------|
| P-256 | 256 бит | ~3072 бит RSA |
| P-384 | 384 бит | ~7680 бит RSA |
| P-521 | 521 бит | ~15360 бит RSA |

**Преимущества ECC:**
- Меньший размер ключа при той же стойкости
- Быстрее вычисления
- Меньше трафика в протоколах

### ГОСТ Р 34.10-2012

Российский стандарт электронной подписи на эллиптических кривых.

| Параметр | Значение |
|----------|----------|
| Кривые | 256 бит, 512 бит |
| Применение | ЭЦП, аутентификация |

```
Рекомендованные параметры кривых:
- id-tc26-gost-3410-2012-256-paramSetA
- id-tc26-gost-3410-2012-512-paramSetA
- id-tc26-gost-3410-2012-512-paramSetB
```

## Обмен ключами

### Diffie-Hellman

Протокол согласования общего секрета по открытому каналу.

```
Alice                           Bob
  |                               |
  |  g^a mod p  →                 |
  |                 ← g^b mod p   |
  |                               |
  K = (g^b)^a = (g^a)^b = g^ab mod p
```

### ECDH (на эллиптических кривых)

Аналог Diffie-Hellman с меньшими ключами и большей производительностью.

:::tip Гибридное шифрование
На практике асимметричное шифрование используется для обмена симметричным ключом, а данные шифруются быстрым симметричным алгоритмом.
:::
