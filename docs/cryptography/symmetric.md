# Симметричное шифрование

Симметричные алгоритмы используют **один ключ** для шифрования и расшифрования.

## Блочные шифры

### AES (Advanced Encryption Standard)

Международный стандарт, принятый NIST в 2001 году.

| Параметр | Значение |
|----------|----------|
| Размер блока | 128 бит |
| Размер ключа | 128 / 192 / 256 бит |
| Раундов | 10 / 12 / 14 |

```python
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

key = get_random_bytes(32)  # AES-256
cipher = AES.new(key, AES.MODE_GCM)
ciphertext, tag = cipher.encrypt_and_digest(b"secret data")
```

### ГОСТ Р 34.12-2015 «Магма»

Российский стандарт (ранее ГОСТ 28147-89).

| Параметр | Значение |
|----------|----------|
| Размер блока | 64 бит |
| Размер ключа | 256 бит |
| Раундов | 32 |

### ГОСТ Р 34.12-2015 «Кузнечик»

Современный российский блочный шифр.

| Параметр | Значение |
|----------|----------|
| Размер блока | 128 бит |
| Размер ключа | 256 бит |
| Раундов | 9 |

## Режимы работы (ГОСТ Р 34.13-2015)

| Режим | Название | Применение |
|-------|----------|------------|
| ECB | Электронная кодовая книга | ❌ Не рекомендуется |
| CBC | Сцепление блоков | Файловое шифрование |
| CFB | Обратная связь по шифротексту | Поточная передача |
| OFB | Обратная связь по выходу | Поточная передача |
| CTR | Счётчик | Диски, параллельная обработка |
| GCM | Счётчик с аутентификацией | TLS, IPsec |

:::warning ECB небезопасен
Режим ECB шифрует одинаковые блоки одинаково, что раскрывает паттерны в данных.
:::
