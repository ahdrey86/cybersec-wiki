# Введение в криптографию

Криптография — наука о методах обеспечения конфиденциальности, целостности и аутентичности информации.

## Основные задачи

| Задача | Описание | Методы |
|--------|----------|--------|
| **Конфиденциальность** | Защита от несанкционированного доступа | Шифрование |
| **Целостность** | Защита от изменения данных | Хеш-функции, MAC |
| **Аутентификация** | Подтверждение подлинности | ЭЦП, сертификаты |
| **Неотказуемость** | Невозможность отрицания действий | ЭЦП |

## Классификация

```
Криптография
├── Симметричная
│   ├── Блочные шифры (AES, ГОСТ МАГМА, Кузнечик)
│   └── Поточные шифры (RC4, Salsa20)
├── Асимметричная
│   ├── RSA
│   ├── Эллиптические кривые (ECC)
│   └── ГОСТ Р 34.10-2012
└── Хеш-функции
    ├── SHA-2, SHA-3
    └── Стрибог (ГОСТ Р 34.11-2012)
```

## Российские стандарты

В России применяются собственные криптографические стандарты, сертифицированные ФСБ:

- **ГОСТ Р 34.12-2015** — блочные шифры «Магма» и «Кузнечик»
- **ГОСТ Р 34.13-2015** — режимы работы блочных шифров
- **ГОСТ Р 34.10-2012** — электронная подпись
- **ГОСТ Р 34.11-2012** — хеш-функция «Стрибог»
