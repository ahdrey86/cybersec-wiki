# Международные стандарты

Глобальные стандарты и регуляции в области информационной безопасности.

## ISO/IEC 27000

Семейство стандартов по системам управления информационной безопасностью (СУИБ).

### ISO/IEC 27001

**Требования к СУИБ** — сертифицируемый стандарт.

**Цикл PDCA:**

```
     ┌─────────────────────┐
     │      Plan           │
     │ Планирование СУИБ   │
     └──────────┬──────────┘
                ↓
┌───────────────┴───────────────┐
│             Do                │
│   Внедрение и эксплуатация    │
└───────────────┬───────────────┘
                ↓
     ┌──────────┴──────────┐
     │       Check         │
     │ Мониторинг и анализ │
     └──────────┬──────────┘
                ↓
     ┌──────────┴──────────┐
     │        Act          │
     │ Улучшение СУИБ      │
     └─────────────────────┘
```

**Ключевые разделы:**

| Раздел | Содержание |
|--------|------------|
| 4 | Контекст организации |
| 5 | Лидерство |
| 6 | Планирование |
| 7 | Поддержка |
| 8 | Эксплуатация |
| 9 | Оценка результативности |
| 10 | Улучшение |

### ISO/IEC 27002

**Практические меры** — 93 контроля безопасности.

**Категории контролей:**

| Домен | Примеры контролей |
|-------|-------------------|
| Организационные | Политики, роли, ответственность |
| Кадровые | Проверка, обучение, увольнение |
| Физические | Периметр, оборудование, носители |
| Технологические | Доступ, криптография, сети |

## PCI DSS

**Payment Card Industry Data Security Standard** — стандарт безопасности данных платёжных карт.

### 12 требований

| № | Требование |
|---|------------|
| 1 | Файрволы для защиты данных карт |
| 2 | Изменение паролей по умолчанию |
| 3 | Защита хранимых данных карт |
| 4 | Шифрование при передаче |
| 5 | Антивирусная защита |
| 6 | Безопасная разработка |
| 7 | Ограничение доступа |
| 8 | Уникальные идентификаторы |
| 9 | Физическая безопасность |
| 10 | Мониторинг и логирование |
| 11 | Регулярное тестирование |
| 12 | Политика ИБ |

### Уровни соответствия

| Уровень | Критерий | Требования |
|---------|----------|------------|
| 1 | >6 млн транзакций | QSA аудит + ASV сканирование |
| 2 | 1-6 млн | SAQ + ASV |
| 3 | 20k-1 млн | SAQ + ASV |
| 4 | менее 20k | SAQ |

## GDPR

**General Data Protection Regulation** — регламент ЕС о защите персональных данных.

### Принципы

| Принцип | Суть |
|---------|------|
| Законность | Правовое основание обработки |
| Ограничение целей | Только заявленные цели |
| Минимизация | Только необходимые данные |
| Точность | Актуальность данных |
| Ограничение хранения | Не дольше необходимого |
| Целостность | Защита от утраты/изменения |
| Подотчётность | Доказательство соответствия |

### Права субъектов

- Право на информацию
- Право доступа
- Право на исправление
- Право на удаление («право быть забытым»)
- Право на ограничение обработки
- Право на переносимость данных
- Право на возражение

### Штрафы

| Категория | Размер |
|-----------|--------|
| Меньшие нарушения | до 10 млн € или 2% оборота |
| Серьёзные нарушения | до 20 млн € или 4% оборота |
